AEC Data Model API: Updated Usage Reporting Dwanill Mehta 8 May 2026 Dwanill Mehta Share What’s New We’ve introduced an update to how usage for the AEC Data Model API is reported, making it easier to understand and manage your data consumption. Usage is now measured as Data In and Data Out, reported in megabytes (MB), providing a more direct and meaningful view of how your applications interact with the API.
