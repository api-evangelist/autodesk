Below you’ll find a recap of all APS blog posts published since our last recap—including major announcements, plenty of upcoming events and DevCon recaps, and practical code samples.</p>
<h2>Announcements</h2>
<h3><a href="https://aps.autodesk.com/blog/building-agentic-ai-whats-new-autodesk-platform-services">Exciting Updates Shared at Autodesk DevCon 2026</a></h3>
<p>We announced plenty of exciting updates to Autodesk Platform Services at DevCon: Expanding access to your data. Releasing Autodesk Public MCPs. Investing in performance improvements. Dive into it all <a href="https://aps.autodesk.com/blog/building-agentic-ai-whats-new-autodesk-platform-services">in this blog</a>.</p>
<h3><a href="https://aps.autodesk.com/blog/new-automation-api-inventor-vs-template">Announcing a New Automation API for Inventor VS Template</a></h3>
<p>We’ve updated the Visual Studio Template to support .NET 8 app bundles that are required in the case of the 2026 Inventor engine. <a href="https://aps.autodesk.com/blog/new-automation-api-inventor-vs-template">Read the announcement</a> to see how the new template makes certain tasks easier.</p>
<h3><a href="https://aps.autodesk.com/blog/navisworks-data-exchange-connector-now-generally-available">Navisworks Data Exchange Connector Is Now Generally Available</a></h3>
<p>We are excited to announce the Navisworks Data Exchange Connector is now generally available for Navisworks Manage 2024, 2025, 2026 and 2027 (coming soon). <a href="https://aps.autodesk.com/blog/navisworks-data-exchange-connector-now-generally-available">Learn what’s new</a> in the GA release.</p>
<h3><a href="https://aps.autodesk.com/blog/forms-template-layout-and-custom-table-api-are-released">Forms Template Layout and Custom Table API Released</a>&nbsp;</h3>
<p>We are pleased to announce the public beta release of the Forms template Layout API. With this new API, you can directly obtain the full definitions of web form. <a href="https://aps.autodesk.com/blog/forms-template-layout-and-custom-table-api-are-released">Get a summary</a> of the key takeaways and find out where you can learn more.</p>
<h3><a href="https://aps.autodesk.com/blog/end-autocad-2021-engine-new-autocad-2027-engine-released">End of AutoCAD 2021 Engine, New AutoCAD 2027 Engine Released</a></h3>
<p>The AutoCAD 2027 engine (Autodesk.AutoCAD+26_0) for Autodesk Platform Services Automation API for AutoCAD brings significant technical updates. Meanwhile, the AutoCAD 2021 engine (Autodesk.AutoCAD+24_0) will be removed from the service. <a href="https://aps.autodesk.com/blog/end-autocad-2021-engine-new-autocad-2027-engine-released">Learn more here</a>.</p>
<h3><a href="https://aps.autodesk.com/blog/buildingconnected-api-enhancements-connected-workflows-forma">BuildingConnected API Enhancements Available in Public Beta</a></h3>
<p>In March, we introduced the availability of the Bidding tool in Forma as beta. Supporting this capability, the same core BuildingConnected Pro API is made available for Forma Bidding tool—and this round of API enhancements <a href="https://aps.autodesk.com/blog/buildingconnected-api-enhancements-connected-workflows-forma">is now available in public beta</a>.</p>
<h3><a href="https://aps.autodesk.com/blog/introducing-new-token-estimator-tool-aps">Introducing the New Token Estimator Tool from APS</a></h3>
<p>The new Token Estimator can be used to estimate how many Flex tokens or cloud credits are needed, per month, for our Premium APIs. <a href="https://aps.autodesk.com/blog/introducing-new-token-estimator-tool-aps">Read the blog</a> to see how it helps you make informed decisions and calculate your monthly API usage.</p>

<h3><a href="https://aps.autodesk.com/blog/custom-properties-fusion">Custom Properties in Fusion</a></h3>
<p>Custom properties let you store additional data in Fusion. Explore the three different workflows that allow you to create properties, link property definition collections to a hub, and set values using an app or service <a href="https://aps.autodesk.com/blog/custom-properties-fusion">in this blog</a>.</p>
<h3><a href="https://aps.autodesk.com/blog/do-you-still-need-controlnet-testing-next-gen-models-viewer-scenes">Testing Next-Gen Models on Viewer Scenes</a></h3>
<p>We’ve showed you how to use Stable Diffusion with ControlNet on Comfy UI to turn Autodesk Viewer scenes into AI-rendered images. Now we’ve tested a new generation of image models to see if they still need still need ControlNet and depth maps. <a href="https://aps.autodesk.com/blog/do-you-still-need-controlnet-testing-next-gen-models-viewer-scenes">Get the answer here</a>.</p>

<p>What if you could drop a full BIM model onto an open-source web map—not as a glTF 3D mesh, but rendered by the actual APS Viewer engine? <a href="https://aps.autodesk.com/blog/putting-revit-map-how-blend-aps-viewer-maplibre">This blog shows you how</a>, with links to a live demo and the complete source code.</p>
<h3><a href="https://aps.autodesk.com/blog/building-custom-mcp-servers-autodesk-platform-services">Building Custom MCP Servers with Autodesk Platform Services</a></h3>
<p>MCPs are a hot topic in Design and Make—but how do you actually build them? <a href="https://aps.autodesk.com/blog/building-custom-mcp-servers-autodesk-platform-services">Learn the best practices</a> for building MCP servers that integrate with Autodesk Platform Services (APS), including code for setup, transport, authentication, and more.</p>
<h3><a href="https://aps.autodesk.com/blog/auto-generate-new-language-versions-your-add">Auto-Generate New Language Versions of Your Add-In</a></h3>
<p>Have an add-in that you want to make available in multiple languages? With the Machine Translation API, you can translate the text resources only when requested by the customer. <a href="https://aps.autodesk.com/blog/auto-generate-new-language-versions-your-add">This blog has the link</a> to the full repo on GitHub.</p>
<h3><a href="https://aps.autodesk.com/blog/files-data-unlocking-data-driven-aec-workflows-aec-data-model-apis">Unlock Data-Driven AEC Workflows with AEC Data Model APIs</a></h3>
